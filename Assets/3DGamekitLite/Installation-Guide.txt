===============================================
3D GAME KIT LITE - INSTALLATION INSTRUCTIONS
===============================================

IMPORTANT: Follow these steps IN ORDER!

===============================================
STEP 1: INSTALL DEPENDENCIES (DO THIS FIRST!)
===============================================

Before importing the package, you MUST install these Unity packages:

1. Open Unity Package Manager:
   Window > Package Manager

2. For EACH package below, click the "+" button (top-left corner) 
   and select "Add package by name..."

3. Copy and paste these package names ONE AT A TIME:

   com.unity.cinemachine
   (Use version 2.10.5 - NOT 3.x)

   com.unity.inputsystem

   com.unity.postprocessing

   com.unity.ai.navigation

   com.unity.textmeshpro

4. Wait for ALL packages to finish installing!
   (Watch the progress bar at bottom of Unity)

5. Make sure there are NO errors in the Console
   (Window > General > Console)

===============================================
STEP 2: IMPORT THE PACKAGE
===============================================

Now that dependencies are installed:

1. In Unity: Assets > Import Package > Custom Package

2. Select the "3DGameKitLite_v1.0.unitypackage" file

3. Click "Import" (import everything)

4. Wait for Unity to import all assets
   (This may take 1-2 minutes)

===============================================
STEP 3: RUN SETUP WIZARD
===============================================

After import completes:

1. Setup Wizard should open AUTOMATICALLY
   - If it doesn't, go to: Tools > 3D Game Kit > Setup Wizard

2. Click through all the setup steps:
   - Cinemachine version check
   - Apply Tags & Layers
   - Apply NavMesh settings

3. Click "Complete Setup"

===============================================
STEP 4: TEST IT!
===============================================

1. Open the Demo scene:
   Assets/3DGamekitLite/Scenes/Demo.unity

2. Press the Play button (top center)

3. Try the controls:
   - WASD: Move
   - Mouse: Look around
   - Space: Jump
   - Left Click: Attack
   - E: Interact
   - ESC: Pause

4. If everything works, you're ready to create!

===============================================
TROUBLESHOOTING
===============================================

Problem: "Cinemachine not found" errors
Solution: You skipped Step 1! Install dependencies first, then reimport.

Problem: Purple/magenta materials
Solution: Make sure you created a URP project (not Built-in RP)

Problem: Setup Wizard doesn't appear
Solution: Check Console for errors. Make sure all dependencies installed.

Problem: Camera won't move
Solution: Check that Input System is set as active input handler
         (Edit > Project Settings > Player > Active Input Handling > Input System Package)
